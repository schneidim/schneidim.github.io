# Eyal Schneid - Executive Portfolio Website A modern, professional executive portfolio website for Eyal Schneid, Fixer & Builder CEO for technology-led organizations. ## Features ✨ **Modern Design** - Dark theme with premium red accents - Fully responsive (mobile, tablet, desktop) - Smooth animations and transitions - Professional typography 📱 **Sections** - Hero with statistics - The Work (6 service areas) - Operating Thesis (Vector of Growth) - Executive Capabilities (6 areas) - Selected Impact Themes - Executive Profile with bio - Speaking & Media - Engagement Models (3 options) - Contact Form ## Setup Instructions ### 1. Add Your Images Create an `images` folder and add your portrait photos: - `images/eyal-portrait.jpg` - Hero section portrait - `images/eyal-portrait2.jpg` - Executive profile section portrait ### 2. Deploy to GitHub Pages This site is ready to be deployed as a GitHub Pages site: 1. Go to your repository Settings 2. Scroll to "GitHub Pages" 3. Select your branch (main) as the source 4. Your site will be live at `https://schneidim.github.io` ### 3. Connect Your GoDaddy Domain 1. Go to GoDaddy DNS settings 2. Add a CNAME record pointing to `schneidim.github.io` 3. Wait 24-48 hours for DNS to propagate 4. Add your custom domain in GitHub Pages settings ## File Structure ``` . ├── index.html # Main website structure ├── styles.css # Styling with CSS variables ├── script.js # Interactivity and animations ├── images/ │ ├── eyal-portrait.jpg │ └── eyal-portrait2.jpg └── README.md # This file ``` ## Customization ### Change Colors Edit the CSS variables in `styles.css`: ```css :root { --dark-bg: #0f1419; /* Main background */ --accent-red: #ef4444; /* Primary accent color */ --blue-accent: #3b82f6; /* Secondary accent */ --text-primary: #f5f5f5; /* Main text */ --text-secondary: #b0b0b0; /* Secondary text */ } ``` ### Update Content Edit `index.html` to update: - Section titles and descriptions - Statistics in the hero - Work items, capabilities, and engagement models - Profile information - Contact form fields ## Browser Support - Chrome (latest) - Firefox (latest) - Safari (latest) - Edge (latest) - Mobile browsers ## Contact Form The contact form currently logs submissions to the browser console. For production use, connect it to a backend service like: - Formspree - Netlify Forms - SendGrid - Custom backend API ## Performance - Optimized for fast loading - No external dependencies (pure HTML/CSS/JS) - Responsive images - Smooth scroll behavior - Intersection Observer for animations ## License All content © 2025 Eyal Schneid.
